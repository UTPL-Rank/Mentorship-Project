Guía para registrar un nuevo acompañamiento

Registrar los acompañamientos a tiempo nos permite identificar posibles problemáticas en los estudiantes de nuevo ingreso.

Primero accedemos al Panel de control, si no has iniciado sesión anteriormente, o es la primera vez que inicias sesión visita la siguiente guiá

Registrar un acompañamiento

Como mentor, tienes acceso en la barra lateral un botón para registrar un acompañamiento. Pero también puedes encontrar en mis estudiantes el botón para registrar un nuevo acompañamiento.

¿Cómo llenar la ficha de acompañamiento?

Una vez hayas ingresado a la ficha "Registrar Acompañamiento" puedes empezar a llenar los campos solicitados.

  1. Selecciona un estudiante de la lista, con el cual realizaste el acompañamiento, cada acompañamiento se registra únicamente en el perfil de cada estudiante. Si no encuentras el estudiante llena la siguiente ficha como datos incorrectos.
  2. Selecciona el bimestre, si realizaste el acompañamiento durante el primer o segundo bimestre.
  3. Selecciona un tipo de seguimiento, ya sea virtual o presencial.
  4. Selecciona la problemática encontrada o tratada durante el acompañamiento. Si no hay una problemática puedes seleccionar "Ningún Problema". y en el caso de que no encuentres la problemática puedes crear una nueva problemática, pero recomendamos usar una de las opciones predeterminadas.
  5. Llena la información de los temas desarrollados, la descripción del problema y las soluciones planteadas.
  6. En los archivos anexos puedes cargar documentos de word o capturas de pantalla, como evidencia de que el acompañamiento se realizó.
  7. Si este acompañamiento necesita seguimiento o ser derivado a otra instancia marca la casilla de acompañamiento importante.
  8. Inicialmente guarda el acompañamiento y le llegará una notificación al estudiante para que realice la validación correspondiente (recuerdale que revise el spam o acceda al panel).
    Nota: Una vez guardado el acompañamiento no podrás editarlo.
